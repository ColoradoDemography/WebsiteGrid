SDO Website Updates 9/1/2023

Accessibility updates

The SDO Website has been updated to implement the accessibility standards specified in House Bill 21-1110. This law directs all State of Colorado meet the Web Content Accessibility Guidelines (WCAG) standards by June 2024.

The typography of the SDO webpage has been modified to improve page legibility. The header structure of each page has been enhanced to improve page navigation. Under the hood improvements have been made to facilitate screen readers and other accessibility hardware.

Each page has been modified to provide a "Skip to main content" link. Press the tab key after a page has loaded to reveal a link to the main content of the page.

Additional changes coming to the website include revisions to SDO reports and documents to make them more accessible to users utilizing assistive technology, such as a screen reader or a Braille keyboard

Revised Data Lookup Applications

The SDO website features a set of data lookup applications for each major section of the website, Population, Births, Deaths and Migration, Economy and Jobs, and Housing. These applications provide users with the ability to generate and download customized data tables. The lookup applications have been updated to match the layout of the current website and provide more functionality. The on-screen tables use "Data Tables," an open-source table library that allows for sorting and pagination. These applications allow data file customization and data downloads.

Revised Net Migration Dashboards and Lookup Applications

SDO has updated several applications showing net migration data:

  • The Net Migration by Age Dashboard has been updated to include Decennial Census data from 1970 to 2020. County and Regional data lookup applications providing Net Migration data by Age have also been added to the "Birth, Deaths & Migration" lookups page.
  • The Migration by Age Map displays county net-migration data by 5-year age group the 1970's to 2010's. This applicaiton uses data from the Net Migration Patterns for US Counties Dashboard maintined by the Applied Population Lab at the University of Wisconsin-Madison
  • The Colorado State to State Migration Map is a new State-to State migration map that combines migration data from the American Community Survey and migration data provided by the Internal Revenue Service

Updates to SDO's 2022 Jobs, Population Estimate and Forecast data series will be published at the State Demography Summit on November 3, 2023. Updates to American Community Survey data will be completed in January 2024. See the 2022-2023 Data Release Schedule for more details.
